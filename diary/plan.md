# 주니어 AI 개발자 로드맵: 컴퓨팅 사고부터 머신러닝까지 🚀

이 로드맵은 단순한 코딩 학습을 넘어, 컴퓨팅 사고력을 기반으로 AI 시대의 핵심 인재로 성장하기 위한 전문 준비 과정입니다. 초등 고학년의 눈높이에 맞춰 AI를 학습 파트너로 삼아 Python부터 Rust까지, 그리고 머신러닝의 원리까지 깊이 있게 탐험합니다.

## 🎯 학습 목표

*   **AI 활용 능력**: AI 어시스턴트를 활용하여 Python, Rust 등 프로그래밍 언어를 효율적으로 학습하고 코드를 개선하는 능력을 기릅니다.
*   **컴퓨팅 사고력 증진**: 다양한 문제 해결 과제를 통해 추상화, 자동화, 디버깅 등 핵심적인 컴퓨팅 사고력을 체계적으로 훈련합니다.
*   **AI/ML 기초 확립**: 머신러닝의 기초 지식과 LLM(거대 언어 모델)의 배경, 원리, 발전 방향을 이해하여 기술의 본질을 파악합니다.
*   **데이터 리터러시**: 데이터의 중요성을 이해하고, 데이터 수집, 정제, 관리, 분석의 기본 과정을 학습합니다.
*   **협업 및 공유 문화**: Git과 GitHub를 통해 학습 과정과 프로젝트 결과물을 기록, 공유하며 다른 학습자와 소통하고 협업하는 방법을 익힙니다.
*   **실전 경험**: 엄선된 소규모 교육 프로젝트를 수행하며 이론을 실제 문제 해결에 적용하는 경험을 쌓습니다.

## 🗺️ 학습 로드맵 (총 6개월+)

### Phase 1: 컴퓨팅 사고와 프로그래밍 기초 (1~2개월)

프로그래밍의 기반이 되는 컴퓨터 과학의 기본 원리와 문제 해결 능력을 기르는 단계입니다.

*   **1주차: 컴퓨터 과학자와의 첫 만남**
    *   컴퓨터의 작동 원리 (하드웨어, 소프트웨어, 운영체제)
    *   프로그래밍이란 무엇인가? (컴퓨터에게 명령을 내리는 방법)
    *   **AI와 함께하기**: Gemini 같은 AI에게 "컴퓨터는 어떻게 생각해?" 질문하기

*   **2주차: 컴퓨팅 사고력 훈련**
    *   문제 분해, 패턴 인식, 추상화, 알고리즘 설계
    *   **과제**: '학교 가는 길 찾기'를 순서도로 그려보기, 복잡한 그림을 기본 도형으로 분해하기

*   **3주차: Python과 첫 만남 & 개발 환경 설정**
    *   Python의 특징과 활용 분야 (쉽고 강력해요!)
    *   VS Code 설치 및 Gemini Code Assist 확장 프로그램 설정
    *   `print("Hello, AI!")`로 첫 코드 실행 및 AI에게 코드 설명 요청하기

*   **4주차: Git으로 나의 학습 여정 기록하기**
    *   Git & GitHub의 필요성 (버전 관리와 협업)
    *   Repository 생성, `commit`, `push` 로 학습 내용 기록 및 공유 시작

*   **5-8주차: Python 프로그래밍 기초 마스터**
    *   **문법**: 변수, 자료형, 연산자, 조건문(if), 반복문(for, while)
    *   **자료구조**: 리스트, 딕셔너리, 튜플의 개념과 활용
    *   **함수**: 코드 재사용을 위한 함수 만들기
    *   **AI 활용**: "1부터 100까지 더하는 함수 만들어줘" 요청 후, AI가 만든 코드를 분석하고 더 효율적인 방법 토론하기
    *   **미니 프로젝트**: 숫자 맞추기 게임, 간단한 계산기 만들기

### Phase 2: 데이터와 AI의 세계로 (3~4개월)

데이터를 다루는 방법과 머신러닝의 기본 원리를 학습하여 AI 개발의 초석을 다집니다.

*   **9-10주차: 데이터의 중요성과 수집**
    *   데이터란 무엇이며 왜 중요한가?
    *   웹 스크레이핑 기초 (BeautifulSoup, requests 라이브러리)
    *   **프로젝트**: 관심 있는 주제의 뉴스 기사 제목 크롤링하여 수집하기

*   **11-12주차: 데이터 분석 및 시각화 입문**
    *   Pandas 라이브러리로 데이터 정제 및 가공하기
    *   Matplotlib, Seaborn으로 데이터 시각화하기
    *   **프로젝트**: 서울시 공공데이터(따릉이 이용 현황 등)를 분석하고 그래프로 표현하기

*   **13-14주차: 머신러닝 기초 원리 탐험**
    *   머신러닝이란? (지도학습, 비지도학습)
    *   Scikit-learn 라이브러리로 간단한 예측 모델 만들어보기 (예: 붓꽃 품종 분류)
    *   **AI 활용**: "머신러닝의 '학습'은 컴퓨터가 어떻게 하는 거야?" 원리 질문하기

*   **15-16주차: LLM의 세계**
    *   인공지능, 머신러닝, 딥러닝, LLM의 관계
    *   트랜스포머 모델과 어텐션 메커니즘의 기본 개념
    *   **탐구 활동**: ChatGPT와 Gemini는 어떻게 탄생했을까? 발전 과정과 원리 조사하여 정리하기

### Phase 3: 시스템 프로그래밍과 고급 주제 도전 (5개월 이후)

더 높은 성능과 안정성이 요구되는 분야를 위한 언어인 Rust를 학습하며 하드웨어에 대한 이해를 높입니다.

*   **17-20주차: Rust 시작하기 - 안전하고 빠른 언어**
    *   Rust의 철학: 소유권(Ownership), 빌림(Borrowing), 생명주기(Lifetimes)
    *   Python과 Rust의 차이점 비교하며 학습하기
    *   **AI 활용**: Python 코드를 Rust 코드로 변환 요청하고, 차이점 설명 듣기
    *   **미니 프로젝트**: 간단한 커맨드 라인(CLI) 프로그램 만들기 (예: 파일 내용 검색 도구)

*   **21-24주차: Rust 중급 및 미니 프로젝트**
    *   구조체(Structs), 열거형(Enums), 트레잇(Traits)
    *   오류 처리, 스마트 포인터
    *   **프로젝트 아이디어**:
        *   간단한 블록체인 원리 구현
        *   멀티스레딩을 이용한 동시성 프로그래밍 맛보기
        *   텍스트 기반의 간단한 어드벤처 게임 제작

## ✅ 학습 원칙 및 성장 가이드

1.  **오류는 성장의 기회**: 오류 메시지는 '틀렸다'는 표시가 아니라 '해결할 문제'라는 신호입니다. AI에게 오류 메시지를 보여주고 원인과 해결책을 물어보며 디버깅 능력을 키우세요.
2.  **질문을 두려워하지 않기**: "Why?"라는 질문을 항상 가지세요. '왜 이 코드는 이렇게 작동할까?', '왜 이 방법이 더 효율적일까?' AI는 훌륭한 질문 파트너가 되어줄 것입니다.
3.  **결과물 공유하기**: 매주 학습한 내용과 프로젝트 결과물을 GitHub에 꾸준히 `commit`하고 `push`하세요. 나의 성장 기록이자 최고의 포트폴리오가 됩니다.
4.  **함께 성장하기**: 다른 친구들의 GitHub를 방문해 코드를 읽어보고, 질문하거나 좋은 점을 칭찬해주세요. 함께 배우고 성장하는 경험은 무엇보다 값집니다.
5.  **꾸준함이 실력**: 매일 정해진 시간에 꾸준히 학습하는 습관을 만드세요. 하루 1시간의 집중이 주말에 몰아서 하는 것보다 훨씬 효과적입니다.
