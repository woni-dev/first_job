# 초등학생을 위한 컴퓨터와 파이썬 코딩 공부 계획 🚀

컴퓨터와 코딩을 처음 만나는 초등학생 친구들을 위한 즐거운 학습 계획표입니다. 어려운 용어보다는 재미있는 놀이처럼, 차근차근 따라 하다 보면 어느새 멋진 코딩 탐험가가 되어 있을 거예요!

## 🎯 학습 목표

*   컴퓨터와 인터넷을 올바르고 안전하게 사용할 수 있다.
*   코딩의 기본 원리(순서, 반복, 조건)를 놀이처럼 이해한다.
*   파이썬이라는 프로그래밍 언어로 간단한 프로그램을 직접 만들 수 있다.
*   문제를 스스로 해결하려는 생각하는 힘을 기른다.

---

## 🗺️ 학습 로드맵 (총 12주)

### 1단계: 컴퓨터와 친해지기 (1~2주차)

컴퓨터라는 신기한 기계와 먼저 친구가 되어보는 시간이에요.

*   **1주차: 컴퓨터 탐험대!**
    *   컴퓨터는 무엇으로 이루어져 있을까? (본체, 모니터, 키보드, 마우스)
    *   파일과 폴더는 왜 필요할까? (내 방 서랍 정리하기에 비유)
    *   타자 연습 게임으로 키보드와 친해지기

*   **2주차: 신나는 인터넷 세상!**
    *   인터넷으로 무엇을 할 수 있을까? (정보 검색, 영상 보기 등)
    *   안전하게 인터넷 사용하는 방법 배우기 (개인정보 보호, 나쁜 댓글 안 달기)
    *   궁금한 것을 검색하는 방법 익히기 (예: "가장 큰 공룡은?")

### 2단계: 코딩 준비 운동, 블록 코딩! (3~4주차)

글자 대신 블록을 조립하며 코딩의 원리를 배워요. (스크래치, 엔트리 추천)

*   **3주차: 블록 코딩이랑 인사하기**
    *   스크래치(또는 엔트리) 가입하고 화면 구경하기
    *   고양이 캐릭터(스프라이트) 움직여보기
    *   "안녕!" 말풍선 띄워보기

*   **4주차: 나만의 이야기 만들기**
    *   배경과 다른 캐릭터 추가하기
    *   캐릭터끼리 대화 주고받게 만들기
    *   클릭하면 소리가 나거나 색깔이 바뀌게 만들기 (이벤트와 반복 개념)

### 3단계: 진짜 코딩의 세계로! 파이썬 기초 (5~8주차)

이제 진짜 프로그래밍 언어인 파이썬을 만날 시간이에요!

*   **5주차: 파이썬과 첫 만남**
    *   파이썬은 왜 배울까? (쉽고 강력해요!)
    *   컴퓨터에 파이썬 설치하기 (또는 웹 기반 파이썬 사이트 활용)
    *   `print("Hello, Python!")`으로 첫 코드 실행하기

*   **6주차: 파이썬 기본 문법**
    *   숫자와 글자를 컴퓨터에 저장하기 (변수)
    *   컴퓨터랑 덧셈, 뺄셈 놀이하기 (산술 연산)

*   **7주차: 컴퓨터에게 생각 가르치기**
    *   "만약 ~하면 ~해!"라고 명령하기 (조건문 if)
    *   간단한 퀴즈 게임 만들기 (예: "우리나라의 수도는?")

*   **8주차: 반복 작업은 컴퓨터에게!**
    *   "열 번 반복해!"라고 시키기 (반복문 for)
    *   이름 10번 출력하기, 1부터 10까지 더하기

### 4단계: 파이썬으로 만드는 미니 프로젝트 (9~12주차)

지금까지 배운 내용으로 재미있는 프로그램을 직접 만들어요.

*   **9주차:** 숫자 맞추기 게임 만들기
*   **10주차:** 간단한 계산기 만들기
*   **11주차:** 터틀 그래픽으로 그림 그리기 (거북이 로봇으로 도형 그리기)
*   **12주차:** 나만의 프로젝트 아이디어 내고 만들어보기 (예: 자기소개 프로그램, 간단한 동화 만들기)

## ✅ 학습 체크리스트

각 주차별 목표를 달성했는지 체크하며 학습을 진행해 보세요!

### 1단계: 컴퓨터와 친해지기
*   **1주차**
    - [ ] 컴퓨터 구성 요소(본체, 모니터 등)를 안다.
    - [ ] 파일과 폴더를 만들고 정리할 수 있다.
    - [ ] 타자 연습 게임을 해봤다.
*   **2주차**
    - [ ] 인터넷으로 정보를 검색할 수 있다.
    - [ ] 안전한 인터넷 사용 규칙을 배웠다.

### 2단계: 코딩 준비 운동, 블록 코딩!
*   **3주차**
    - [ ] 스크래치(또는 엔트리)에 가입했다.
    - [ ] 캐릭터를 움직여봤다.
    - [ ] 캐릭터가 말풍선으로 말하게 해봤다.
*   **4주차**
    - [ ] 새로운 배경과 캐릭터를 추가해봤다.
    - [ ] 캐릭터들이 서로 대화하는 이야기를 만들었다.
    - [ ] 클릭했을 때 소리가 나거나 색깔이 바뀌게 만들었다.

### 3단계: 진짜 코딩의 세계로! 파이썬 기초
*   **5주차**
    - [ ] 파이썬을 실행할 환경을 준비했다.
    - [ ] `print()`를 사용해 화면에 글자를 출력했다.
*   **6주차**
    - [ ] 변수를 사용해 숫자나 글자를 저장했다.
    - [ ] 파이썬으로 덧셈, 뺄셈 등 계산을 해봤다.
*   **7주차**
    - [ ] `if`를 사용해 조건을 만드는 법을 배웠다.
    - [ ] 간단한 퀴즈 게임을 만들어봤다.
*   **8주차**
    - [ ] `for`를 사용해 명령을 반복하는 법을 배웠다.
    - [ ] 반복문을 이용해 이름 여러 번 출력하기 등을 해봤다.

### 4단계: 파이썬으로 만드는 미니 프로젝트
- [ ] **9주차:** 숫자 맞추기 게임을 완성했다.
- [ ] **10주차:** 간단한 계산기를 완성했다.
- [ ] **11주차:** 터틀 그래픽으로 원하는 도형을 그렸다.
- [ ] **12주차:** 나만의 작은 프로젝트를 완성했다.
